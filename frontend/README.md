# SynergyScope Frontend

Professional web interface for SynergyScope analytics platform.

## Structure
- static/css/ - Stylesheets
- static/js/ - JavaScript modules
- templates/ - HTML pages

## Technologies
- D3.js for network graphs
- Chart.js for charts
- Vanilla JavaScript (no frameworks)

## Development
Open index.html or run with backend server.
